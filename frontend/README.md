# Stillwater Pulse Frontend

Next.js 16 application for displaying Instagram posts via RSS feeds.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Configuration

Edit `data/feeds.json` to configure which Instagram accounts to display. Each account should have a valid RSS feed URL (e.g., from rss.app or fetchrss.com).
